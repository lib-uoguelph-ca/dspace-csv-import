The Atrium: Bulk importing

Introduction

This directory is set up in a modified DSpace Simple Archive Format. It can be populated with files and metadata and then imported into the Atrium.

Files

Each item has its own numbered directory, e.g. item_123/. Place files for an item into the directory.

Note that items can contain multiple files. For more information on the logical structure of The Atrium contact Carol Perry.

Metadata

For each item, update the dublin_core.csv file that should already be in the directory. This can be done using Excel, but be careful not to change the file format when you save it.

Additional metadata lines may be added to the CSV file as long as the elements are supported by the Atrium. Some elements can be included multiple times. An element that is often repeated is dc.subject, which is used for keywords. To include an element multiple times simply copy the appropriate row, paste a copy into the CSV document, and modify the value as necessary.

Language

The dc.language.iso metadata value should be set to the code for the main language of an item. Other metadata tags may be in a different language, e.g. there could be a dc.title.alternate tag that includes a French title. If this is the case, please add the appropriate language code to the Language column of that tag.

English and French language codes are as listed below. For other languages, please refer to the 639-1 column (two-character codes) on the ISO 639 Wikipedia page.

ISO language codes for English and French

en
English
fr
French

Submitting

Once all of the files are in place and the metadata has been set up please zip the entire directory up and send it back.
